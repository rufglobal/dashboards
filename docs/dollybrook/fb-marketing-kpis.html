What We've Learned — 53 Days of Data

What Drives Results

  • Community question threads — avg score 52/post. Sauglasville community Q drove 34K page views
  • Neighbor intro in local groups — SH Chatline: 196R, 30S from one post
  • Guest Guide solicitation — businesses react to their own mentions
  • Small/mid groups 6-80K — 0.4-0.66% engagement vs 0% in 200K+ groups
  • ISO reply pipeline — 5 leads, 1 converted. Greater MI STR = #1 source
  • Pool + fire content — Pool avg score 32 (#1), firepit 26.6
  • Marginal fit replies — bystander effect is real (confirmed Apr 22)
  • Descriptive, informative comments — Facebook rewards detail

What Doesn't Work

  • Listing-style posts in community groups — avg score 1/post
  • Large groups 200K+ — declined or zero engagement every time
  • Links in post body — Facebook throttles reach. First comment only.
  • Cold groups without prior warming — 100% decline rate
  • Skipping marginal-fit ISOs — bystander audience exists, always reply
  • Re-uploading reels — native share only, never paste link
  • One-liner ISO replies — descriptive, detailed replies perform better
